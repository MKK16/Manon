# ProductManagement REST API

Developed RESTful service that can retrieve product and price details by ID.

## Technology Stack

Below is a list of technologies used in the development.

```bash
Java 8
FireBase - Nosql
Maven
SpringBoot 2.x
Pivotal Cloud Foundry
```
